## üêõ Troubleshooting

### If a service fails to build:

**Check the logs:**
- Click on the failed service
- Click "Logs" tab
- Look for error messages

**Common issues:**
- Missing environment variable ‚Üí Add it in Environment tab
- Build timeout ‚Üí Retry deployment
- Port issues ‚Üí Render auto-assigns ports

### If database won't connect:

- Check that PostgreSQL service is running
- DATABASE_URL is auto-generated by Render
- Wait for PostgreSQL to be fully started before backend
